Frequently asked questions

How to use it?

Press the button, choose the picture and wait for magic. If you are not happy with the results you can change the resolution or color palette.

Why this app?

All bad good ideas are born in the pub. My good friend after few beers asked me if I can create mobile app that will convert picture from camera to cross stitch pattern.
So I said, "Go home. You Are drunk! Brilliant idea! I will give You 10% of all the money i make from such app".

Is there a mobile version of this app?

Yes there is. Just visit this site with your mobile device and you can make cross stitch patterns directly from your camera.

Why it is'nt working on my Internet Explorer?

Because this application is using HTML5 fetaures that doesn't work on Internet Explorer.
Thre truth is that I was lazy to make it work on Internet Explorer. Sometimes it is really painfull to make things work on Internet Explorer. This is a project only for fun so I tried not to kill fun with painfull Internet Explorer compatibilty.

Will it work on Internet Explorer in the future?

Definitely, maybe.

This application is rubish! Is there any other app that does the same?

No! Definitely isn't.
